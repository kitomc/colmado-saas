# ColmadoAI — Documentación del Proyecto

## 📁 Estructura de este repositorio de documentación

```
colmadoai_docs/
├── AGENTS.md              ← REGLAS DEL AGENTE (OpenCode lee esto primero)
├── PROMPT_MAESTRO.md      ← Cómo iniciar cada sesión de OpenCode
├── POC_ColmadoAI_SaaS.md  ← Documento técnico completo del sistema
└── checklists/
    ├── fase1.md           ← Nano-tareas: Schema y CRUD básico (Convex)
    ├── fase2.md           ← Nano-tareas: Webhook WhatsApp + LLM
    ├── fase3.md           ← Nano-tareas: Telegram Bot
    ├── fase4.md           ← Nano-tareas: App Android + Bluetooth
    └── fase5.md           ← Nano-tareas: Panel Web Admin
```

## 🚀 Cómo empezar

1. Coloca todos estos archivos en la RAÍZ de tu proyecto de código.
2. Abre OpenCode en ese directorio.
3. Copia el **Prompt de inicio de sesión** de `PROMPT_MAESTRO.md`.
4. Reemplaza los valores según tu situación actual.
5. Pégalo al inicio de la sesión de OpenCode.
6. Sigue el orden de nano-tareas sin saltear ninguna.

## ⚠️ Reglas de oro

- **NUNCA** pidas más de una nano-tarea a la vez.
- **SIEMPRE** verifica el checklist antes de pasar a la siguiente tarea.
- **SIEMPRE** inicia cada sesión con el Prompt Maestro.
- Si algo falla, pega el error exacto y usa el Protocolo de Corrección.
